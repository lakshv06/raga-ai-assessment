# RaagAI Frontend Assessment — React + TypeScript + Vite

This repository contains the solution for the RaagAI frontend developer assessment. It demonstrates a small React + TypeScript application scaffolded with Vite, implementing the assessment's required UI and interactions.

## Features

- Assessment UI and interaction flows implemented as requested
- Built with React and TypeScript for type-safe components
- Fast development experience via Vite
- Responsive layout with basic accessibility considerations

## Tech Stack

- React
- TypeScript
- Vite
- CSS

## Quick Start

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```
